# 📝 Todo-List

A web-based task management app built with HTML, CSS and vanilla JavaScript.

## ✨ Features

- ➕ Add a task with a title and details
- 📅 Assign a due date to each task
- 🏷️ Categorize tasks : **Work**, **Personal**, **Urgent**
- ⚠️ Visual alert — tasks due today **blink in yellow**
- 📋 Add a detailed description to each task

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## 🚀 Getting Started

No installation required — simply open `index.html` in your browser.
